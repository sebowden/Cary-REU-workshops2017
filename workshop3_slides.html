Plotting continuous data

When we want to look at the relationship between two continuous variables, we plot our data using a scatterplot. Here, well look at the relationship between mosquito development time and mosquito body size to answer the question: Is there a relationship between body size and development time in larval mosquitoes?

H0: There is no relationship between these variables.
Ha: There is a (positive/negative) relationship between these variables.

Plotting continuous data

First, lets make a basic scatterplot of these two variables. To do this, we call the base R function plot. This function needs at least two arguements, in the form of plot(x.axis.variable,y.axis.variable).

plot(mosq_data$Dev_time, mosq_data$Abd_length)

Plot customization

There are many ways to customize plots in R. Were going to run through the most basic options. First, lets add informative axis labels. To label the x-axis, add an arguement within plot using xlab="axis title". To label the y-axis, add an arguement within plot using ylab="axis title". Be sure to include relevant units of measure in your axis labels.

plot(mosq_data$Dev_time, mosq_data$Abd_length, xlab = "Development time (days)", 
    ylab = "Abdominal length (mm)")

Plot customization

Next, lets add a title to the plot. To do this, add an arguement within plot using main="Plot title". Plot titles should be informative and concise.

plot(mosq_data$Dev_time, mosq_data$Abd_length, xlab = "Development time (days)", 
    ylab = "Abdominal length (mm)", main = "Relationship between body size and development time")

Plot customization

If you find the text on your plot difficult to read, you can change the size of the axis labels (cex.lab), plot title (cex.main), and/or the numbers on each axis (cex.axis) by adding these arguements within plot. The arguement cex is short for character expansion, so a value of 1.5 means you want the text to be 1.5 times larger than the default size. cex can be less than one if you want to reduce the text size, or more than one if you want to increase the text size.

plot(mosq_data$Dev_time, mosq_data$Abd_length, xlab = "Development time (days)", 
    ylab = "Abdominal length (mm)", main = "Relationship between body size and development time", 
    cex.lab = 1.5, cex.main = 1.7, cex.axis = 1.5)

Plot customization

You can also change the color, shape, and/or size of the points in a scatterplot. To change the color of your points, add an arguement within plot using color="color name". To see a (long) list of all the color names that R knows, type colors() into your console.

plot(mosq_data$Dev_time, mosq_data$Abd_length, xlab = "Development time (days)", 
    ylab = "Abdominal length (mm)", main = "Relationship between body size and development time", 
    col = "red")

Plot customization

To change the shape of your points, add an arguement within plot using pch= followed by a number between 0-25. pch is short for point character. To see all of the pch options in R, type ?pch into your console. I chose pch=17, which is a solid triangle.

plot(mosq_data$Dev_time, mosq_data$Abd_length, xlab = "Development time (days)", 
    ylab = "Abdominal length (mm)", main = "Relationship between body size and development time", 
    col = "red", pch = 17)

Plot customization

To change the size of your points, add an arguement within plot using cex= followed by your character expansion factor. This works the same way as cex.axis and others we used a few slides ago, where a number less than one will shrink your point size and a number greater than one will increase your point size.

plot(mosq_data$Dev_time, mosq_data$Abd_length, xlab = "Development time (days)", 
    ylab = "Abdominal length (mm)", main = "Relationship between body size and development time", 
    col = "red", pch = 17, cex = 1.5)

Plot customization

Finally, you might want to have more than one plot in a single window. To do this we just have to make a small change to the plot parameters. Before we call our plot, we add a line of code: par(mfrow=c(number of rows, number of columns)). This essentially allows you to create grids of plots all in the same window.

# specify (n.rows, n.columns)
par(mfrow = c(1, 2))  #one row, two columns

plot(mosq_data$Dev_time, mosq_data$Abd_length, xlab = "Development time (days)", 
    ylab = "Abdominal length (mm)", main = "Relationship between body size and development time", 
    pch = 17, col = "red", cex = 1.5)

plot(mosq_data$Abd_length, mosq_data$Dev_time, xlab = "Abdominal length (mm)", 
    ylab = "Development time (days)", main = "Relationship between body size and development time", 
    pch = 17, col = "blue", cex = 1.5)

Statistical tool #1: linear regression

Now that we know how to get our data into R and visualize it, its time to make inferences from our data. The way we do this with two continuous variables is through linear regression.

We can perform a linear regression in R using the function lm(), which is short for linear model. lm() needs at least one arguement a formula. Formulas in R relate a dependent variable (y) to one or more independent variables (x). Formulas are specified as lm(dependent.variable~independent.variable). Each variable is usually a column from a dataset or a single vector. lm() also takes an arguement that specifies the dataset youre working from in the form of data=data.set.name.

my.model <- lm(Abd_length ~ Dev_time, data = mosq_data)

Statistical tool #1: linear regression

Now that we have specified our linear model, we can ask R for a summary of the model using summary(my.model).The output from summary() will print several statistics. When youre working with a single independent variable, you should focus on the R-squared and associated p-value to draw conclusions.

summary(my.model)

## Multiple R-squared:  0.05596,    Adjusted R-squared:  0.0557 
## F-statistic: 215.8 on 1 and 3641 DF,  p-value: < 2.2e-16

R-squared tells us the amount of variation in the dependent variable (y) that is explained by the independent variable (x). R-squared values can range from zero (no relationship) to one (perfect linear relationship).

In this case, our R-squared is 0.05596. This means that development time (our independent variable) explains 5.596% of the variation in body size in our dataset. While this isnt a particularly large R-squared, it is statistically significant (p-value < 2.2e-16), meaning this relationship is unlikely to be due to random chance.

Combining plots and stats

Now that we have our statistical results, we can combine them with our figure. Well add a line over the top of our scatterplot that represents the linear model (using abline() function), as well as add text to the plot to display the R-squared and p-value (using text() function).

# don't forget to reset your graphical parameters
par(mfrow = c(1, 1))

plot(mosq_data$Dev_time, mosq_data$Abd_length, xlab = "Development time (days)", 
    ylab = "Abdominal length (mm)", main = "Relationship between body size and development time", 
    pch = 17, col = "red")

# adds a line over the scatterplot
abline(my.model)

# adds text at location (x,y) on plot
text(13, 0.17, labels = "R-squared = 0.05596")
text(13, 0.16, labels = "p-value = 2.2e-16")
