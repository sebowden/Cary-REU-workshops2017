Implementing a t-test in R

Now we want to know if there is a statistically significant difference in size between male and female mosquitoes. This is where we have to carefully consider our alternative hypothesis.

Ha: There is a difference in size between females and males. == two-sided t-test

Ha: Females are (larger OR smaller) than males. == one-sided t-test

When in doubt, go with a two-sided t-test because it is more conservative.

# for a two-sided test
t.test(Abd_length ~ Sex, data = mosq_data, alternative = "two.sided")
## 
##  Welch Two Sample t-test
## 
## data:  Abd_length by Sex
## t = 6.8374, df = 3180.8, p-value = 9.628e-12

## 95 percent confidence interval:

# for a one-sided test
t.test(Abd_length ~ Sex, data = mosq_data, alternative = "greater")
##

Using the p-value and means printed above, how would you interpret these results? Remember, a t-test will only tell you if there is a difference between two group, not the direction of the difference. Always pair the results of a statistical test with a plot of your data for interpretation!

Visualizing differences between more than two groups

Now, lets combine these statistics into a single figure. First, well download and install the package plotrix which contains the fuction we need to use to plot error bars plotCI.

Within the function plotCI(), we first need to specify x (point locations along the x-axis) and y (point locations along the y-axis; in the case, these are the means for each species). Then we add arguements for the width of the error bars (uiw=) and the direction of the error bars (err="y"). The next arguement add=FALSE tells R that we want to make a new plot. The remaining arguements (pch, col, cex, xlab, ylab, and main) are all arguements that weve worked with before. The final arguement xaxt="n" tells R that we dont want it to plot the x-axis. Finally, we use the legend() function to add (you guessed it!) a legend.

# this installs and loads a package into our workspace. you only need to
# install a package once, but you'll have to load (i.e., library()) a
# package each time you restart RStudio.

# install.packages('plotrix')
library(plotrix)

plotCI(x = c(1:3), y = c(CQ.mean, AL.mean, AG.mean), uiw = c(CQ.sd, AL.sd, AG.sd), 
    err = "y", add = FALSE, pch = 16, col = c("red", "blue", "purple"), cex = 1.2, 
    xlab = " ", ylab = "Development time (days)", main = "Differences in larval development time", 
    xaxt = "n")

# the first arguement specifies the location of the legend (e.g.,
# 'bottomleft','topright'); legend is a vector of labels; col is a vector of
# colors associated with each label; pch set the point character to a closed
# circle to match the plotted points; bty specifies that we don't want to
# draw a box around the legend
legend("topright", legend = c("CQ", "AL", "AG"), col = c("red", "blue", "purple"), 
    pch = 16, bty = "n")

ANOVA

Next, lets run an ANOVA to determine if any of these groups differ significantly from the others.

ANOVA

Now that we know that development time is not the same across all species, we need to figure out which species (one or more) differs from the rest. We do this using post-hoc pairwise t-tests between each pair of species, correcting for the number of tests we perform (i.e., Bonferroni correction)

pairwise.t.test(mosq_data$Dev_time, mosq_data$Species, p.adj = "bonf")
